## To install react app

### Usual Option: 

``` bash
npx create-react-app app_name

# Runs the server
npm start
```

### Using Vite: 

``` bash
# defines the file structure
npm create vite@latest app_name
cd app_name
# installs the dependencies
npm install
# runs the server
npm run dev
```
